Basic PointsGrid

We create a PointsGrid which generates for us an array of points [x, y]. Then we set up our SvgRenderer. You can choose if you want to use an existing svg in DOM or to create a new one. In the end we render our PointsGrid. 

import { PointsGrid, SvgRenderer } from 'svg-gen';

const pointsGrid = new PointsGrid({});

const renderer = new SvgRenderer({
    svgElementSelector: '#example-basic>svg'
});

for (const row of pointsGrid.pointRows) {
    for (const point of row) {
        renderer.createCircle(point, 2);
    }
}

Points Grid random distortion and points on edge

const pointsGrid = new PointsGrid({
    randomDistortionX: 4,
    randomDistortionY: 2,
    maxX: 50,
    maxY: 50,
    rows: 6,
    cols: 4,
});

const renderer = new SvgRenderer({
    svgElementSelector: '#example-with-options>svg',
        viewBox: [0, 0, 50, 50],
});

for (const row of pointsGrid.pointRows) {
    for (const point of row) {
        renderer.createCircle(point, 1);
    }
}

Points Grid with Options

We can modify many things. Here are some possible options. We also set the viewBox to the size of the points range, so the svg content will be scaled to the size of its svg element. 

const pointsGrid = new PointsGrid({
        randomDistortionX: 4,
        randomDistortionY: 2,
        maxX: 50,
        maxY: 50,
        rows: 6,
        cols: 4,
    });
    
    const renderer = new SvgRenderer({
        svgElementSelector: '#example-with-options>svg',
            viewBox: [0, 0, 50, 50],
    });
    
    for (const row of pointsGrid.pointRows) {
        for (const point of row) {
            renderer.createCircle(point, 1);
        }
    }
